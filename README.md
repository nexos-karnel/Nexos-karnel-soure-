«Lightweight ARM Bare-Metal Operating System

---

🇮🇩 Tentang

NexOS adalah sistem operasi bare-metal berbasis ARM yang dikembangkan dari nol menggunakan bahasa C dan Assembly.

Proyek ini dibuat untuk mempelajari bagaimana sebuah sistem operasi bekerja mulai dari proses booting, kernel, manajemen memori, interrupt, driver perangkat keras, hingga networking.

---

🇬🇧 About

NexOS is an ARM bare-metal operating system built from scratch using C and Assembly.

The project focuses on learning operating system internals, low-level programming, kernel development, hardware drivers, and networking.

---

✨ Features

- ARM Bare-Metal Kernel
- Bootloader & Startup Code
- Interrupt Handling (IRQ)
- UART Driver
- Ring Buffer
- VirtIO Network Driver
- VirtIO Block Driver
- DHCP Client
- DNS Resolver
- TCP Stack
- Basic File System
- Modular Kernel Architecture
