# 🚀 Vision

NexOS aims to become a modern, lightweight, modular, and educational operating system built completely from scratch using **C** and **Assembly**.

The project focuses on learning low-level programming, kernel development, hardware interaction, and operating system design while remaining open source and community-driven.

---

# 🎯 Development Goals

Our primary goals are:

- Build a stable ARM bare-metal kernel
- Keep the source code modular and maintainable
- Support real ARM hardware
- Improve networking capabilities
- Develop a complete memory management system
- Introduce multitasking
- Expand hardware driver support
- Build a complete operating system ecosystem
