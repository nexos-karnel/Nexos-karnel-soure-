# 🇮🇩 Penjelasan Lisensi

NexOS menggunakan lisensi **GNU General Public License v3.0 (GPL-3.0)**.

Lisensi ini bertujuan menjaga agar NexOS tetap menjadi proyek **Open Source** yang dapat dipelajari dan dikembangkan bersama.

---

# ✅ Yang Boleh Dilakukan

- ✅ Menggunakan source code NexOS.
- ✅ Mempelajari cara kerja kernel.
- ✅ Memodifikasi source code.
- ✅ Membuat fork repository.
- ✅ Mengirim Pull Request.
- ✅ Membagikan ulang source code sesuai GPL-3.0.

---

# ❌ Yang Tidak Boleh Dilakukan

- ❌ Menghapus copyright yang ada pada source code.
- ❌ Mengklaim seluruh proyek NexOS sebagai karya sendiri.
- ❌ Menyebarkan versi modifikasi tanpa mematuhi ketentuan GPL-3.0 jika mendistribusikannya.
- ❌ Menggunakan nama atau logo NexOS dengan cara yang menyesatkan seolah-olah itu adalah proyek resmi.

---

# 🤝 Open Source Philosophy

NexOS dibuat untuk belajar, berbagi, dan berkembang bersama.

Jika Anda memperbaiki bug, menambahkan fitur, atau meningkatkan performa, kami sangat menghargai kontribusi tersebut.
